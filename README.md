# rethink-digitalocean-webui
All of my startups are using DigitalOcean as cloud service provider. But as a person who uses their webui every single day, I'm sooooo tired of it. It has a really bad UX design, and it's slow + lack of some management features I need (eg. firewall notes notes, and batch editing). SO I'm building my own version of DigitalOcean's webui with AI!
